# 뒤집기

- 풀이 유형: 그리디

- 풀이 idea: 0과 1이 바뀔 때마다, 끊어서 각 배열의 수를 증가시키자?

EX1

```
0001100
[000, 00] -> 2개
[11] -> 1개
=> 1번 뒤집으면 됨
```

EX2

```
11111
[] -> 0개
[11111] -> 1개
=> 0번
```

EX3

```
00000001
[0000000] -> 1개
[1] -> 1개
=> 1번
```

EX4

```
11001100110011000001
[00,00,00,00000] -> 4개
[11,11,11,11,1] -> 5개
=> 4번
```
